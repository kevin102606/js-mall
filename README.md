# js-mall 

### 项目说明：

一个PC端商城

### 项目技术栈：   

+ html
+ css 
+ javascript

### 项目功能：

+ 主页面
+ 登录页面
+ 注册页面
+ 分类页面
+ 详情页面
+ 购物车页面
